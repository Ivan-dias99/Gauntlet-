# MERGE GATE

## Merge is allowed only when all are true

1. The change strengthens the living organism in `src/ruberra/`
2. The change does not re-legitimize `src/app/`
3. The change does not introduce generic SaaS, startup, or wrapper gravity
4. The change leaves the repository clearer than before
5. The claimed completion does not outrun structural truth
6. The next move after merge is legible

## Automatic Failure Conditions

- dead surface is presented as primary
- docs contradict mounted runtime reality
- canon drift is unnamed
- cosmetic progress is framed as foundational progress
- wave objective is missing or handoff is absent

## Allowed Verdicts

- MERGE
- REVISE
- RESTRUCTURE
- REJECT

## Final Principle

Momentum is not a merge criterion.
Truth is.
