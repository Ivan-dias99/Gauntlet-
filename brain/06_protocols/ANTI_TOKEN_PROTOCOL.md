# ANTI-TOKEN PROTOCOL

## Law

Do not spend tokens to restate what is already canon unless the restatement creates real delta.

## Forbidden output
- long summaries with no new decision
- repeating approved doctrine as filler
- explaining what is already obvious from repo state
- verbose option spreads when one path is clearly strongest
- ceremonial language pretending to be progress

## Required behavior
- compress aggressively when truth is already known
- speak only where there is delta, risk, conflict, or decision
- prefer decisive synthesis over repeated framing

## Pioneer rule
A pioneer that emits long output without advancing truth, merge state, or execution state is drifting.
