# AGENT MEMORY MODEL

Future Ruberra-native agents should not each keep separate private truths.

## Memory law
Agents share one sovereign memory substrate.
They may have different scopes into it, but not different underlying product truth.

## Required properties
- memory must be attributable
- memory must be stateful
- memory must be queryable by scope
- memory must be promotable toward canon when warranted
- memory revocation must leave reason and lineage

## Agent-specific read pattern
An agent should read:
- relevant canon
- relevant decisions
- relevant protocols
- relevant recent chamber/thread memory

before acting on non-trivial work.

## Forbidden memory pattern
- hidden agent-only truth that contradicts sovereign repo truth
- memory without lineage
- memory without revocation path
- memory treated as raw transcript instead of meaningful substrate
