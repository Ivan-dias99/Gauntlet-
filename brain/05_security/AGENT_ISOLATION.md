# AGENT ISOLATION

## Law
Agents may share truth.
They may not share uncontrolled authority.

## Isolation requirements
- each lane acts within scope
- no silent override of another lane's accepted truth
- no second command center
- no merge residue left behind as someone else's problem

## Shared substrate
Agents should share:
- brain
- canon
- memory substrate
- merge law

## Non-shared right
An agent does not gain the right to redefine product truth just because it can produce output.
