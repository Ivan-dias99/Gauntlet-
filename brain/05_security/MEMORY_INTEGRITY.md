# MEMORY INTEGRITY

## Law
Meaningful action must be able to leave memory.
Meaningful memory must be attributable and resumable.

## Integrity failures
- silent mutation
- orphan memory with no context
- memory rewritten casually
- high-value observation lost because no retention path exists

## Required properties
Memory should be:
- attributable
- stateful
- queryable
- promotable to canon when warranted
- revocable with reason when invalidated

## Ruberra note
Memory is part of architecture, not an accessory layer.
