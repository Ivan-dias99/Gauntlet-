# EXECUTION PERMISSIONS

## Law
No execution should happen without explicit scope.

## Reversible actions
May proceed with normal directive acceptance if scope and acceptance criteria are clear.

## Consequential actions
Require:
- explicit scope
- explicit acceptance criteria
- visible consequence path

## Destructive actions
Require:
- explicit justification
- explicit confirmation path
- no ambiguity in target or effect

## Null consequence rule
If execution cannot be justified, the system should emit null consequence instead of pretending to act.
