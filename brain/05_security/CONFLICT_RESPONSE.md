# CONFLICT RESPONSE

## Law
Conflict should be surfaced, not hidden.

## Repo conflict
If a branch conflicts with `main`, the conflict must be resolved or the branch must be killed.
It may not remain floating as false-progress residue.

## Canon conflict
If two truths compete, stronger repo truth and resolved lineage must decide.
Unresolved ambiguity should not be disguised as harmony.

## Agent conflict
If two lanes disagree, lane authority and accepted repo truth outrank model preference.

## Required response
- name the conflict
- identify the authority boundary
- resolve, harvest, or close
- do not leave residue pretending to be harmless
