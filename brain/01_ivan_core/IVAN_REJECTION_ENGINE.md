# IVAN REJECTION ENGINE

This file defines what should be rejected fast.

## Reject immediately if output is:
- generic SaaS logic
- dashboard symmetry without meaning
- AI wrapper behavior pretending to be product truth
- verbose without delta
- structurally weak but cosmetically polished
- fragmented into many competing centers
- branch/pr/preview treated as equal to merged `main`
- memory treated as archive furniture
- creation reduced to casual prompting

## Reject unless proven otherwise if output:
- introduces a second authority source
- preserves residue instead of converging it
- keeps stale PRs alive as if harmless
- replaces organism logic with feature pile logic
- flatters category norms instead of transcending them

If a proposal does not absorb real creator pain or increase sovereign creation power, it does not deserve mass.
