# RUBERRA MEMORY LAWS

Memory is not a convenience feature.
Memory is an architectural layer.

## Memory must do
- preserve high-value consequence
- make continuity resumable
- keep observations attributable
- support promotion into canon when warranted
- support revocation with reason when invalidated

## Memory must not become
- archive furniture
- note dump
- generic search list
- detached knowledge app

## State law
Meaningful memory should have state.
At minimum:
- observed
- retained
- hardened
- revoked

## Promotion law
Not all memory becomes canon.
Canon requires stronger pressure than retention.

## Design law
Memory surfaces should feel ledger-like, consequence-bearing, and structurally calm.
Utilities are subordinate to retained matter.
