# RUBERRA CANON

## Canon law

Canon is not opinion.
Canon is retained truth hardened into law.

## Canon must protect
- one organism
- one sovereign source line
- one runtime truth
- chamber distinction
- memory continuity
- consequence closure
- anti-generic posture

## Canon forbids
- preview treated as product truth
- stale PR treated as active authority
- dead surfaces treated as live runtime
- memory reduced to archive furniture
- creation reduced to generic prompting
- shell reduced to dashboard navigation

## Canon mutation law

Canon should mutate only when:
- reality changed
- a stronger truth displaced a weaker one
- contradiction was proven and resolved
- product direction matured in a founder-authorized way

Canon should never mutate because:
- a branch exists
- a preview looks exciting
- a contributor prefers another pattern
- a generic trend is popular
