# Ruberra Transmutation Protocol

## Objective
Convert raw chamber outputs into canonical, reusable, routeable runtime truth.

## Protocol Stages
1. **Capture**: persist user/assistant outputs as runtime objects.
2. **Bind**: attach run to continuity with `linkedObjectId`.
3. **Transition**: mark lifecycle status with timestamps.
4. **Consequence**: emit signals/rewards/transfers.
5. **Ledgerize**: expose in Profile (work, memory, exports, recommendations).
6. **Index**: include in cross-surface search index.

## Recovery Rules
- Keep governance kernel, event consequence, continuity provenance.
- Remove ornamental naming and duplicate governance sources.
- Keep abstractions only when they produce runtime consequence.
